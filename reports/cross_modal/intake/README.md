# Intake Reports

This directory holds dataset-specific intake reports produced during Phase 1.

The expected flow is:

1. create or update the candidate manifest
2. complete the dataset worksheet in `docs/cross-modal/`
3. record actual ingest findings here
4. update the manifest and registry once the dataset is accepted or rejected
